# Omarchy Royal Theme

![preview](/preview.png)

## Quick Install (Recommended)

```bash
omarchy-theme-install https://github.com/notSagyo/omarchy-royal-theme
```

## Manual Installation

1. Clone the repository:

```bash
git clone https://github.com/notSagyo/omarchy-royal-theme.git ~/.config/omarchy/themes/royal
```

2. Switch to the theme using the Omarchy menu:
   - Press `Super + Shift + Alt + Space` choose "Royal"
